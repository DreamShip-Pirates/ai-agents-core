# Skill: Log Analyzer

**Purpose**: Scans logs for fatal patterns and leaks.

## Scan Targets
- **Process Suicide**: Identify unexpected application crashes.
- **OOM**: Detection of Out-of-Memory errors.
- **Credential Leaks**: Ensure no API keys or secrets are printed in plain text.
- **Silent Failures**: Correlate muted errors with system state.

## Usage
Used by the **Observability Engineer** and **Fixer**.
