# Shared Skill Toolbox

This directory contains placeholders for backend-specific skills.

- **api_contract_validator**: Scans routes for OpenAPI/Swagger compliance.
- **silent_failure_hunter**: Detects swallowed exceptions.
- **security_threat_modeler**: Automated security checklists.
- **log_analyzer**: Analyzes server logs for error patterns.
- **migration_safety_check**: Verifies SQL/Firestore migrations.
- **cost_efficiency**: Tracks DB operation counters.
